# 📊 Macroeconomic Forecasting & Dashboard Project

## 🚀 Overview
This project analyzes and forecasts key macroeconomic indicators including CPI (inflation), GDP, and unemployment using time-series modeling and data visualization.

---

## 🎯 Objectives
- Analyze historical economic trends
- Forecast inflation using statistical and ML models
- Build an interactive dashboard for insights

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Machine Learning (Linear Regression)
- Time Series (ARIMA)
- Power BI

---

## 📂 Dataset
- CPI (Consumer Price Index)
- GDP (Gross Domestic Product)
- Unemployment Rate

---

## 🔧 Data Processing
- Handled missing values
- Converted GDP quarterly → monthly
- Aligned time-series data
- Feature engineering (lag, rolling, inflation metrics)

---

## 📈 Models Used
- Linear Regression (with lag features)
- ARIMA model for time-series forecasting

---

## 📊 Dashboard Features
- KPI cards (CPI, GDP, Unemployment)
- Trend analysis charts
- Inflation (YoY) visualization
- Interactive date filtering

---

## 🔍 Key Insights
- Inflation shows long-term upward trend
- GDP reflects steady economic growth
- Unemployment spikes during economic crises
